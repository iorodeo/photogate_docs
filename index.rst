Project Documentation
================================================

Documentation for the `Educational photogate kit for Arduino (EDU-01) <http://www.iorodeo.com/content/educational-photogate-kit-arduino>`_. 

.. figure:: _static/main_image.png
   :align:  center


The photogate kits include the following parts:

* Photogate shield for Arduino
* Pair of photogates: pre-assembled and ready to use
* Pair of photogate cables (7 ft long RJ11 cables)
